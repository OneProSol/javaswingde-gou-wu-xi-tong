### 一、相关技术
- 前端：`javaswing`。
- 后端：`java`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 浏览商品（结算、加入购物车）

管理员功能：1. 登录、2. 商品类别添加、3. 商品类别维护、4. 商品添加、5. 商品维护
